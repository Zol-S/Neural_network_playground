Convolutional layers
In this interactive tutorial, convolutional and pooling layers can be linked together. First select the type of layer, the type of predefined 3x3 kernels (only in case of convolutional layer) or the size of kernel (only in case of pooling layer), select the value of the stride, then click on Add layer button, to insert the layer to the end of the network. After adding at least one layer, click on any of the sample input images to send it to the network of layers.
